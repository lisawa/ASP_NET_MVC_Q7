# ASP_NET_MVC_Q7

建立一個 ASP NET MVC 專案，實作 TODO List 功能

## 功能說明

1. 新增 TODO 的項目
2. 改變 TODO 項目的狀態，可修改為 Active 或 Completed
3. 可篩選 Active, Completed 或全部的項目
4. 可刪除 TODO 項目
5. 可清除 Completed 的項目

## 資料庫設計

1. 請自行設計資料表結構，將 Create Table schema 寫在 docs/schema.sql 裡
2. 請使用本機資料庫自行測試，不需要提交整個資料庫檔案

## 備註

1. 合理的資料處理流程
2. 合理的程式碼架構分層


## 進階

1. 使用 Vue 前端框架實作 UI
